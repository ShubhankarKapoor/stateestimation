# Distribution System State Estimation Research Code

This repository contains open-source research software developed as part of a PhD
and subsequent peer-reviewed publications on distribution system state estimation.

## Purpose
The code provides accessible implementations of established distribution system
state estimation methods. It has been used to generate results for
peer-reviewed papers and a doctoral thesis. The focus is on transparency,
reproducibility, and reuse, rather than production-grade deployment or the
development of new algorithms.

## Repository Status
This repository reflects its origins as a research codebase. Multiple branches are
maintained to preserve research traceability for different studies and experiments.
While the core methods are validated through peer-reviewed research, the code
currently has limited documentation and a research-oriented structure.

Improving code organisation, documentation, examples, and overall reusability is
the primary motivation for the proposed Open CER Tools & Data Enablement project.

These branches are preserved for reproducibility and research reference.

## Intended Audience
The primary audience includes researchers, distribution network service providers
(DNSPs), and other stakeholders seeking transparent baseline implementations of
distribution network estimation and power flow methods.
